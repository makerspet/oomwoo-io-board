# oomwoo I/O + Controller Board (RK3562 + STM32G070)

Schematic (PDF) for the **[oomwoo](https://github.com/makerspet/oomwoo)** open-source robot
vacuum's controller + I/O board: a Rockchip **RK3562** application processor (ROS 2 / Nav2 /
SLAM) paired with an **STM32G070** for real-time motor and sensor I/O — plus the vacuum
peripherals: drive-wheel motors, main/side brushes, suction fan, water pump, LiDAR, MIPI
camera, IMU, IR cliff/proximity + bumper sensors, battery/BMS, WiFi/BT, eMMC and LPDDR4.

📄 **[oomwoo-io-board-RK3562-schematic.pdf](oomwoo-io-board-RK3562-schematic.pdf)**

> **Status: early reference schematic**, published for review and discussion.
> It has **not been validated** — do **not** fabricate from it as-is. The editable source
> design (Altium) and a verified, manufacturable revision will follow.

Part of the [oomwoo](https://github.com/makerspet/oomwoo) project — an open-source, ROS 2,
3D-printable robot vacuum you build yourself. Come say hi on
[Discord](https://discord.gg/3y2JKz5T25).
